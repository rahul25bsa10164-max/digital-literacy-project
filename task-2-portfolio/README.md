# Task 2: Build Your Student Digital Portfolio

### Platforms Chosen
For this task, I have established my professional presence on **GitHub**, **LinkedIn**, and **ResearchGate**. 

### Purpose of Each Platform
* **GitHub:** I use this as my technical repository to showcase my coding projects, security scripts, and digital literacy documentation.
* **LinkedIn:** This is my primary networking tool. I use it to connect with industry professionals, showcase my academic achievements at VIT Bhopal, and build a professional brand.
* **ResearchGate:** As a student specializing in Cyber Security and Digital Forensics, I chose this platform to follow academic papers and stay updated with the latest research in the security field.

### Four-Year Plan
Over the next four years, I plan to use GitHub to build a timeline of my technical growth. I will use LinkedIn to actively search for internships and ResearchGate to stay connected with the academic side of digital forensics. Having these three platforms integrated ensures I am industry-ready by graduation.
